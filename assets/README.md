# 動画・画像素材置き場

## ファイル構成
- hero.mp4         → Hero 動画（推奨: H.264, 1080p, 30fps）
- hero.webm        → Hero 動画 WebM版（モバイル最適化）
- hero-poster.jpg  → 動画読み込み前に表示するサムネイル

## 動画スペック推奨
- 解像度: 1920×1080 以上
- 長さ: 10〜30秒（ループ再生）
- 容量: 10MB 以内が理想（ページ速度のため）
- 音声: なし（autoplay muted のため）
- トリミング: バーガーを焼く・肉汁・盛り付けシーン が◎
